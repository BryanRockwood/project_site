The Project

So What are we doing here?

This project is my version of the good old "3 Month Coding bootcamp" path that many developers go through when they are looking to make a career change. My plan is to continually improve and add to this site as I learn more topics around frontend (front-end?) development. I want this to be a sort of "living" website where it evolves as I do as a developer. Right now it's close to a blank canvas with some scribbles on it. In time i'll add some color to those scribbles, and eventually connect all those lines to make a more coherent picture. In a perfect version of this, I'll send people to this website to be my portfolio of work, and serve as a visual representation of my growth in this frontend journey.
