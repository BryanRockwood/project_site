Player Comparison tool

I'm a big fantasy football guy. I love talking about the game, the players, and how unfair of a game it ultimately is. Below is one of the main tools I've wanted to create to help not only myself, but to potentially help other people getting into the weeds of fantasy football. Here is the evolving progression of my player comparison tool.


First, I have a small scrpit that takes in two players and compares them against each other. At the moment I am simply using an array of values to serve as our data, and this tool searches the array for the two names and compares their "ranking" (read: index in the array) and returns the lower value of the two. Since rankings we usually want the "Number 1" option, we are looking for the lower number of the indices. This initial version contains just a few names to test out functionality.
